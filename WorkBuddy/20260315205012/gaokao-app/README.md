# 高考志愿帮

一款面向中国高考考生的智能志愿填报助手应用，帮助考生根据分数、选科和意向专业快速查找合适的院校。

## 功能特性

### 1. 分数查大学
输入高考分数，自动匹配可报考的院校
- 支持按省份筛选
- 支持按批次筛选（本科一批、本科二批、专科）
- 显示预估录取概率和专业组信息
- 提供冲、稳、保三种策略建议

### 2. 选科中心
查看各选科组合可报考的专业范围
- 12种主流选科组合全覆盖
- 显示专业覆盖率
- 按分数段推荐可报考院校
- 列出受限专业和选科建议

### 3. 专业库
浏览各专业类别的详细信息
- 12个学科大类
- 包含专业介绍、就业方向、薪资待遇
- 推荐开设该专业的优质院校

### 4. 招生章程
查看院校招生章程要点
- 录取规则
- 身体要求
- 收费标准
- 联系方式

## 数据说明

- 高校数据：涵盖全国2800+所高校（本科+专科）
- 分数线：基于2024-2025年各省录取数据
- 专业数据：包含12个学科大类、90+个专业类
- 选科数据：覆盖12种选科组合的专业覆盖率

## 技术栈

- React 18 + TypeScript
- Vite + Tailwind CSS
- shadcn/ui 组件库
- 响应式设计，支持移动端

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

1. **分数查大学**：在首页输入高考分数，选择省份和批次，点击查询即可获得可报考院校列表
2. **选科中心**：选择自己的选科组合，查看可报考的专业范围和推荐院校
3. **专业库**：按学科门类浏览专业，了解各专业的就业前景
4. **招生章程**：查看各院校的招生政策要点

## 注意事项

- 本应用数据仅供参考，实际录取情况请以官方招生信息为准
- 分数线每年会有波动，建议结合历年数据综合分析
- 志愿填报需谨慎，建议咨询专业老师或使用官方志愿填报系统
